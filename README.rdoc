== README

This is a blog app using AngularJs and rails following the tutorial from @asanderson15 his repo is located here: https://github.com/asanderson15/rails-angular-tutorial. 
His tutorial is located here: http://asanderson.org/posts/2013/06/03/bootstrapping-angular-rails-part-1.html. 

I work in a Haml shop and decided to change a few things from the source to allow the views to be written in Haml, the rest of the code should be almost spot on to the original.
